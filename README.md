# 📖 Overview

This project demonstrates an exploratory and statistical analysis of the **ToothGrowth** dataset, a well-known experimental dataset available in R.

The analysis simulates a real-world research scenario where a biological laboratory evaluates how different **Vitamin C supplements** and **dosage levels** affect tooth growth in guinea pigs.

Through exploratory data analysis, statistical summaries, sampling techniques, and visualizations, the project transforms experimental observations into meaningful insights that support evidence-based conclusions.

---

# 🎯 Business / Research Problem

Researchers need to determine whether different Vitamin C delivery methods and dosage levels produce significantly different outcomes in tooth growth.

Without proper statistical analysis, it is difficult to understand:

- How dosage influences growth.
- Whether one supplement performs better than another.
- How consistent the experimental results are.
- The variability within each treatment group.

This project applies statistical techniques to answer these questions through data analysis.

---

# 🎯 Project Objectives

The main objectives of this project are:
